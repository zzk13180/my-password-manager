# Passbox

A simpler way to manage your sensitive information and links.

## Getting Started

install [nodejs](https://nodejs.org) and [pnpm](https://pnpm.js.org/)

```shell
pnpm i
```

```shell
pnpm dev
```

```shell
pnpm build
```
